Who am I?

Backend-heavy full stack developer with 2+ years of experience building secure, scalable web applications. I create clean, responsive interfaces using HTML, CSS, JavaScript, jQuery, Bootstrap, Tailwind CSS, and Vue.js, and develop robust backends with PHP frameworks like Laravel, CodeIgniter, Yii, and Symfony, working with MySQL, MariaDB, and SQLite databases. My work includes designing RESTful APIs and microservices, implementing high-performance caching and session management with Redis, and building asynchronous, event-driven workflows using RabbitMQ for reliable message queuing. I containerize applications with Docker and deploy on AWS, all while following industry best practices for security and maintainability. With a strong grasp of data structures, algorithms, OOP, design patterns, and SOLID principles, I focus on writing clean, efficient code, and I’m experienced with software testing, CI/CD pipelines, Agile workflows, and tools like Git and Linux to keep projects running smoothly.
